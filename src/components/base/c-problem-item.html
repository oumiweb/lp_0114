データ、どんどん増えていませんか？

写真、動画、顧客データ、研究記録…
年率30%のペースで増え続けるデータ。
数年で倍増する量をどう保存しますか？

外付けハードディスクを増やし続ける？
機器の故障でデータが消失する危険性があります。
クラウドの料金が膨らみ続ける？
多くのクラウドでは月次/年次の固定費用がどんどん大きくなります。
クラウドを月次契約したが
使っていない領域がある？
契約済みのクラウドで使っていない領域は無駄になってしまいます。
2

海外クラウドには
アクセスのリスクがあります

クラウドサービス、本当に安全ですか?
「大手だから安心」と思っていませんか?

実は、多くのクラウドサービスは海外の
サーバーにデータを保存しています。

  • 各国の政府の主張や（国家安全保障等）によってアクセスできなくなるリスク。
  • 世界的な規範が存在しないことで不信感が増大している地政学的リスクや法規制の変更により、お客様の大切なデータに危険性が迫ります。
（（出典元：総務省：デジタルインフラ（DC等）整備に関する有識者会合より）
3

特にこのような方は対策が急務です

以下の方々は特に注意が必要です。

大量の写真・動画を保存するクリエイター
作品の消失・改ざんは創作活動の致命的な損失となります。
大量の研究データを保存する
必要がある研究者
大量の研究成果が消失・改ざんされれば、学術的信頼性が失われます。
海外クラウドに不安を感じている企業
機密情報の流出・消失・改ざんは、企業の競争力と信頼を失墜させます。
